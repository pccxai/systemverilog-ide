# Code of Conduct

This repository follows a contributor conduct policy focused on respectful, professional collaboration.

Expected behavior:
- Use welcoming and precise language.
- Keep feedback focused on the work.
- Respect maintainers' review decisions and repository scope.

Unacceptable behavior includes harassment, threats, doxxing, and sustained disruptive conduct.

Maintainers may moderate issues, pull requests, discussions, or access when needed to protect the project. Report urgent private concerns to security@pccx.ai.
